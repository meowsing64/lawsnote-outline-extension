## 中文版

### 資料收集

本擴充功能**不收集、不儲存、不傳送**任何使用者的個人資料或瀏覽資料。

### 運作方式

本擴充功能僅在 Lawsnote（lawsnote.com、atlas.lawsnote.com）的判決頁面上執行，功能為解析頁面上已公開的判決書文字結構，並在頁面左側顯示導航大綱。所有處理皆在使用者的瀏覽器本機完成。

### 權限說明

本擴充功能不要求任何特殊瀏覽器權限（如存取瀏覽紀錄、Cookie、書籤等）。

### 網路連線

本擴充功能**不會發送任何網路請求**。不連接任何外部伺服器、不使用分析工具、不包含廣告。

### 第三方服務

本擴充功能不使用任何第三方服務或 SDK。

### 變更通知

如本隱私權政策有所變更，將於本頁面更新。

## English Version

### Data Collection

This extension does **not collect, store, or transmit** any personal data or browsing information.

### How It Works

This extension operates only on judgment pages of Lawsnote (lawsnote.com, atlas.lawsnote.com). It parses the publicly available judgment text structure on the page and displays a navigation outline on the left side. All processing is performed locally in the user's browser.

### Permissions

This extension does not require any special browser permissions (such as access to browsing history, cookies, bookmarks, etc.).

### Network Activity

This extension **does not make any network requests**. It does not connect to any external servers, does not use analytics tools, and does not contain advertisements.

### Third-Party Services

This extension does not use any third-party services or SDKs.

### Changes

If this privacy policy is updated, changes will be reflected on this page.
